# HTML入门学习

- ==Hyper Text Market Language==(超文本标记语言)

- 优势：浏览器普遍支持，跨平台，市场需求

- W3C标准

  > 万维网联盟（World Wide Web Consortium）
  >
  > 结构化标准语言（HTML，XML）
  >
  > 表现标准语言（CSS）
  >
  > 行为标准（DOM、ECMAScript）


## 页面结构分析

| 元素名  |                     描述                     |
| :-----: | :------------------------------------------: |
| header  | 标题区域的内容（用于页面或页面中的一个区域） |
| footer  |               标记脚区域的内容               |
| section |           Web页面中的一块独立区域            |
| article |                独立的文章内容                |
|  aside  |       相关的内容和应用（常用于侧边栏）       |
|   nav   |                导航类辅助内容                |

## 表单

- 所有的input表单元素都需要有name属性，加以区分表单的值

> hidden 隐藏表单
>
> disable 不可用
>
> readonly 只读

##  表单初级验证

- 为了减少无效数据的请求

>placeholder : 提示信息
>
>required : 要求不能为空
>
>pattern : 字符串检验 正则表达式
